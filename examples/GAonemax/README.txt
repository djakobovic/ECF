
OneMax example - illustration for BitString genotype (see more on this example in help/tutorial.html).


Running the example (more info in help/install.html)
	Windows: compile the provided VisualStudio solution in 'VS' directory (having previously compiled the ECF_lib solution)
	Linux: compile with the provided 'Makefile' (having previously compiled the ECF library)

Description
    The onemax optimization is a simple search which counts the ones in a bit string. The optimal solution is a string consisting entirely of ones.

Parameters (in the config file)

        for BitString genotype, only its size must be defined
